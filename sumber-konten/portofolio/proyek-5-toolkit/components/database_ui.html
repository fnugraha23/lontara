Database & Kriteria Editor

Pengaturan Parameter Acuan

Kelola kriteria syarat tumbuh tanaman dan standar klasifikasi lahan yang tersimpan dalam format JSON.

Kesesuaian Lahan

Syarat tumbuh tanaman (LREP II / PPT Bogor).

Kemampuan Lahan

Kriteria hambatan fisik (Arsyad / USDA).

Kebutuhan Pupuk

Tabel kadar hara & koefisien serapan tanaman.

Editor Kriteria: kesesuaian.json

Ubah ambang batas parameter untuk klasifikasi otomatis.

Parameter Satuan S1 (Optimal) S2 (Cukup) S3 (Marginal) Aksi
Temperatur ($tc$) °C
Curah Hujan ($wa$) mm/thn
Struktur Data JSON

Data disimpan dalam skema *Key-Value Pair* untuk mendukung integrasi dengan Python/Machine Learning di masa depan. 

{
  "tanaman": "Padi Sawah",
  "kriteria": {
    "tc": { "S1": [22, 28], "S2": [18, 22], "S3": [15, 18] },
    "ph": { "S1": [6.0, 7.0], "S2": [5.5, 6.0], "S3": [4.5, 5.5] }
  }
}
Keamanan Data

Perubahan di sini akan langsung mempengaruhi logika perhitungan pada seluruh modul LONTARA.

Pastikan format angka menggunakan titik (.) sebagai desimal.
