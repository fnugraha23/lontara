Toolbox ArcGIS Desktop: Otomasi Kelas Lereng

Sebuah Custom Python Toolbox (.pyt) yang saya kembangkan untuk menyederhanakan dan mengotomatisasi proses ekstraksi kelas lereng dari data DEM. Alat ini dilengkapi dengan algoritma pembersihan data (Minimum Mapping Unit/MMU) yang berjalan secara iteratif.

Lihat Alur Kerja

ArcPy Scripting

Dikembangkan sepenuhnya menggunakan bahasa Python dan library ArcPy untuk performa komputasi geospasial yang maksimal.

Parameter Dinamis

Pengguna dapat menentukan jumlah kelas lereng dan skala peta secara dinamis langsung dari antarmuka Toolbox.

Iteratif MMU

Dilengkapi algoritma eliminasi noise (Minimum Mapping Unit) iteratif otomatis untuk menghasilkan poligon siap kartografi.

Alur Kerja Eksekusi Toolbox

Bagaimana Python Toolbox ini mengubah data mentah menjadi informasi topografi yang terklasifikasi hanya dalam beberapa kali klik.

1

Input Data DEM Mentah

Proses dimulai dengan memasukkan data elevasi mentah (Digital Elevation Model / DEM) ke dalam workspace (seperti DEM wilayah Maros pada contoh ini). Data ini akan menjadi basis perhitungan kemiringan lereng.

2

Eksekusi Python Toolbox Custom

Memanggil alat pemrosesan khusus "Analisis Kelas Lereng.pyt" melalui ArcCatalog. Tool ini menampilkan antarmuka yang bersih untuk memasukkan direktori Input DEM, Output Raster, dan Output Polygon yang diinginkan.

3

Konfigurasi Kelas & Skala Pemetaan

Fitur utama dari skrip ini adalah kemampuannya menerima parameter dinamis. Analis dapat menentukan klasifikasi lereng (misal: 5 Kelas atau 7 Kelas) serta menentukan resolusi skala peta (50K, 100K, 250K) yang nantinya akan memicu algoritma Minimum Mapping Unit (MMU) untuk menyederhanakan poligon secara otomatis.

4

Selesai: Output Hasil Terklasifikasi

Dalam hitungan detik, algoritma Python mengeksekusi konversi kemiringan, reclassify, raster ke poligon, dan eliminasi poligon kecil (MMU). Hasil akhirnya adalah peta lereng terklasifikasi yang akurat, mulus, dan siap digunakan untuk penyusunan laporan atau desain tata ruang.
