LontaraGeo Engine:
Otomasi Analisis Spasial

Implementasi Google Earth Engine API untuk pengolahan citra satelit Sentinel-2 dan Landsat guna mendukung presisi pengelolaan wilayah DAS secara real-time.

Buka Engine Penuh Lihat Demo Interaktif

Metodologi & Inovasi

Sebagai mahasiswa pascasarjana Ilmu Tanah di IPB University, saya mengembangkan platform ini untuk mempercepat analisis spasial dalam riset mengenai Kinerja Sub-DAS Tanralili, Sulawesi Selatan.

Sistem menggunakan Python Flask yang di-host di Koyeb untuk menghubungkan permintaan parameter pengguna langsung ke infrastruktur cloud processing Google. Platform ini mengotomatiskan langkah cloud masking dan kalkulasi indeks vegetasi tanpa memerlukan sumber daya komputasi lokal yang berat.

Fitur Unggulan

  • 01. Cloud-Based Processing: Seluruh komputasi dilakukan di server Google, memungkinkan analisis data besar di perangkat apa pun.
  • 02. Dinamis AOI: Penentuan wilayah riset melalui alat gambar poligon interaktif untuk ekstraksi data spesifik lokasi.
  • 03. Otomasi Indeks: Perhitungan instan untuk NDVI, EVI, dan NDWI untuk pemantauan kesehatan lahan dan sumber daya air.
