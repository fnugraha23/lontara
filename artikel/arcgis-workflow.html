Automasi Workflow ArcGIS untuk Analisis Hidrologi DAS

Oleh Fajar Nugraha · 6 Menit Membaca

Mengekstraksi jaringan sungai dan batas Daerah Aliran Sungai (DAS) dari Digital Elevation Model (DEM) adalah tugas klasik dalam hidrologi. Tanpa otomasi, proses ini dapat memakan waktu yang sangat lama terutama untuk cakupan wilayah yang luas.

Meninggalkan Klik Manual dengan Python

# Pipeline Hidrologi Otomatis
import arcpy
from arcpy.sa import *

arcpy.env.workspace = "C:/Lontara/Project_DAS"
dem_filled = Fill("raw_dem.tif")
flow_dir = FlowDirection(dem_filled, "NORMAL")
flow_acc = FlowAccumulation(flow_dir)
stream_net = Con(flow_acc > 1500, 1)
stream_net.save("river_network.tif")

Otomatisasi menjamin reproduktibilitas. Jika data DEM diperbarui, kita hanya perlu menjalankan kembali skrip yang sama untuk mendapatkan hasil instan.
