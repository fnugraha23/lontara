LontaraGeo Engine:
Otomasi Analisis Spasial

Integrasi Google Earth Engine API untuk pemrosesan citra satelit Sentinel-2 dan Landsat secara real-time guna mendukung pengelolaan Sub-DAS Tanralili yang presisi.

Buka Layar Penuh (Fullscreen) Pelajari Metodologi

Metodologi & Inovasi

Sebagai bentuk dedikasi di bidang ilmu tanah (IPB University), platform ini dibangun untuk memangkas hambatan efisiensi dalam pengolahan data spasial DAS secara signifikan.

Sistem memanfaatkan Python Flask untuk menjembatani parameter input pengguna langsung ke infrastruktur cloud processing Google. Fitur utama mencakup Cloud Masking otomatis dan perhitungan indeks vegetasi (NDVI/EVI).

Python Flask
GEE API

Data Science

Fitur Unggulan

  • 01. Pemrosesan Tanpa Beban: Seluruh komputasi berat dilakukan di server Google secara cloud-based.
  • 02. Dinamis AOI: Penentuan area analisis melalui alat gambar poligon interaktif di peta.
  • 03. Filter Temporal: Memungkinkan perbandingan indeks kesehatan tutupan lahan antar periode waktu secara instan.

Siap Menerapkan Intelijen Spasial?

Integrasikan alur kerja otomatisasi Google Earth Engine ke dalam proyek atau instansi Anda untuk pengambilan keputusan berbasis data yang lebih cepat.
