LontaraGeo Engine:
Otomasi Analisis Spasial

Otomasi pemrosesan citra satelit Sentinel-2 dan Landsat secara real-time melalui Google Earth Engine untuk mendukung pengelolaan Sub-DAS Tanralili yang presisi.

Buka Layar Penuh (Fullscreen) Pelajari Metodologi

Metodologi & Inovasi

Sebagai mahasiswa pascasarjana Ilmu Tanah di IPB University, saya mengembangkan platform ini untuk memecahkan hambatan efisiensi dalam pengolahan data spasial riset saya mengenai Kinerja Sub-DAS Tanralili di Sulawesi Selatan.

Sistem memanfaatkan Python Flask untuk menjembatani parameter input pengguna langsung ke infrastruktur cloud processing Google. Fitur utama mencakup Cloud Masking otomatis dan perhitungan indeks vegetasi (NDVI/EVI) tanpa beban komputasi lokal.

Python Flask
GEE API
Koyeb Cloud
Leaflet JS
Machine Learning

Fitur Unggulan

  • 01. Pemrosesan Tanpa Beban: Seluruh komputasi berat dilakukan di server Google, menjaga performa perangkat pengguna tetap optimal.
  • 02. Dinamis AOI: Penentuan Area of Interest (AOI) melalui alat gambar poligon interaktif untuk analisis lokal yang akurat.
  • 03. Filter Temporal: Memungkinkan perbandingan indeks kesehatan tutupan lahan antar periode waktu secara instan.

Aplikasi WebGIS Interaktif

Siap Menerapkan Intelijen Spasial?

Integrasikan alur kerja otomatisasi Google Earth Engine ke dalam proyek atau instansi Anda untuk pengambilan keputusan berbasis data yang lebih cepat.
