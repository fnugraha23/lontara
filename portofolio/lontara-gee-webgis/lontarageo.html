Cloud Computing Remote Sensing

LontaraGeo Engine:
Otomasi Analisis Spasial

Integrasi Google Earth Engine API untuk pemrosesan citra satelit Sentinel-2 dan Landsat secara real-time guna mendukung pengelolaan Sub-DAS Tanralili yang presisi.

Buka Layar Penuh (Fullscreen) Lihat Demo Di Sini

Metodologi & Inovasi

Sebagai mahasiswa pascasarjana Ilmu Tanah di IPB University, saya mengembangkan platform ini untuk memecahkan masalah efisiensi dalam pengolahan data spasial riset saya mengenai Kinerja Sub-DAS Tanralili di Sulawesi Selatan.

Alih-alih mengunduh citra secara manual, sistem ini memanfaatkan Python Flask untuk menjembatani parameter input pengguna langsung ke cloud processing Google. Algoritma menyertakan Cloud Masking otomatis dan perhitungan indeks vegetasi (NDVI/EVI) untuk memantau kesehatan tutupan lahan secara aktual.

Python Flask
GEE API
Koyeb Cloud
Leaflet JS
Machine Learning

Fitur Unggulan

  • 01. Efisiensi Komputasi: Pemrosesan ribuan hektar data citra satelit dilakukan di server cloud tanpa membebani perangkat lokal.
  • 02. Custom AOI: Fleksibilitas menentukan area analisis (Area of Interest) melalui alat gambar poligon yang interaktif.
  • 03. Monitoring Temporal: Mendukung perbandingan indeks kesehatan lahan antar periode waktu untuk melihat tren perubahan penggunaan lahan.

Aplikasi WebGIS Interaktif

Pilihlah satelit, tentukan rentang waktu, dan gambarlah area pada peta untuk memulai komputasi spasial.

💡 Tips: Gunakan ikon poligon di pojok kiri atas peta
