LontaraGeo Engine:
Otomasi Analisis Spasial

Integrasi Google Earth Engine API untuk pemrosesan citra satelit Sentinel-2 dan Landsat secara real-time guna mendukung pengelolaan Sub-DAS Tanralili.

Buka Layar Penuh (Fullscreen) Pelajari Metodologi

Metodologi & Inovasi

Sebagai mahasiswa pascasarjana Ilmu Tanah di IPB University, saya membangun platform ini untuk memangkas hambatan efisiensi dalam pengolahan data spasial riset saya mengenai Kinerja Sub-DAS Tanralili.

Python Flask
GEE API

Data Science

Fitur Unggulan

  • 01. Pemrosesan Tanpa Beban: Seluruh komputasi dilakukan secara cloud-based di server Google.
  • 02. Dinamis AOI: Penentuan area analisis melalui alat gambar poligon interaktif di peta.
  • 03. Filter Temporal: Memungkinkan perbandingan indeks kesehatan tutupan lahan antar periode secara instan.
