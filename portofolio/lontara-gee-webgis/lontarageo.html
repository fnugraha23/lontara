LontaraGeo Engine:
Otomasi Analisis Spasial

Integrasi Google Earth Engine API untuk pemrosesan citra satelit Sentinel-2 secara real-time guna mendukung pengelolaan Sub-DAS Tanralili yang presisi.

Buka Layar Penuh (Fullscreen) Pelajari Metodologi

Metodologi & Inovasi

Sebagai mahasiswa pascasarjana Ilmu Tanah di IPB University, saya mengembangkan platform ini untuk memecahkan hambatan efisiensi dalam pengolahan data spasial riset saya mengenai Kinerja Sub-DAS Tanralili.

Python Flask
GEE API
Koyeb Cloud
Leaflet JS
Machine Learning

Fitur Unggulan

  • 01. Pemrosesan Tanpa Beban: Seluruh komputasi berat dilakukan di server Google secara cloud-based.
  • 02. Dinamis AOI: Penentuan Area of Interest melalui alat gambar poligon interaktif untuk analisis lokal.
  • 03. Filter Temporal: Memungkinkan perbandingan indeks kesehatan lahan antar periode secara instan.
