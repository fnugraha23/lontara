Dataset Batas Administrasi

Akses data spasial poligon resmi bersumber dari BIG TASWIL 2024. Data telah divalidasi topologi untuk menjamin kualitas analisis geospasial Anda.

1. Batas Wilayah Provinsi

Dataset ini mencakup batas wilayah administratif tingkat pertama di seluruh Indonesia. Data merujuk pada metadata Batas Wilayah (TASWIL) dengan SRS ID berupa SRGI 2013. Sangat direkomendasikan untuk pemetaan skala nasional dan analisis makro antar provinsi.

Unduh SHP Provinsi (.shp)

2. Batas Wilayah Kabupaten & Kota

Data ini dihasilkan melalui proses Geoprocessing Dissolve dari batas desa untuk menghasilkan unit administratif tingkat II. Sifat data saat ini masih indikatif (sementara), mengacu pada kenampakan alami dan buatan sebagai batas pemetaan awal untuk kebutuhan survei komparasi.

Unduh SHP Kabupaten/Kota (.shp)

3. Batas Wilayah Kecamatan

Dataset tingkat kecamatan menyediakan detail poligon yang bersih dari overlap antar wilayah. Data dikembangkan melalui integrasi Pusat Pemetaan Batas Wilayah (PPBW) BIG dan diperbarui sesuai standar Update Data Tahun 2024 untuk kebutuhan perencanaan teknis daerah.

Unduh SHP Kecamatan (.shp)

4. Batas Wilayah Desa & Kelurahan

Dataset dengan tingkat kedetailan tertinggi, mengintegrasikan data batas wilayah administrasi desa dari Peta Rupa Bumi Indonesia (RBI). Mencakup tabel atribut lengkap seperti Kode Dagri dan Nama Wilayah untuk pemetaan partisipatif dan pemberdayaan ekonomi desa.

Unduh SHP Desa/Kelurahan (.shp)

Spesifikasi Teknis:
Proyeksi: WGS 84 (EPSG:4326) | SRS: SRGI 2013 | Sumber: BIG – TASWIL 2024 | Status: Topologi Clean
